# Bolides
HTML5 recreation of the classic arcade game Asteroids.

## [Play the game](http://nichosta.github.io/bolides/)

## Controls
* Up: accelerate
* Down: slow down
* Left/Right: turn
* SPACE: fire a bullet
* _WASD also works._
